Rocky Tunnel Tutorial 

 

a) Making the shapes

 Select the 2D Quad shape tool. Make some kind of square in the top view. Select the 2D line and make the tunnel path in the Top view two. Once done, for each shape, right click on it, select properties in the context menu and enable 'Draw spline' check box. That'll give us the yellow spline line. Adjust the path vertex if needed.

 

b) Extruding

 Select the Modifier Plugin. In the Path goup, select the quad as the 'shape' and the tunnel path as the path. The mesh is extruded. Now, we have to tweak it for making a real tunnel. First, check 'Splined path' and change path steps up to 10.

That should give you something like that.

Let's tweak it a little more. We use a splined shape to with a step of 3 to have a circle section.

Move the Camera to the tunnel entrance to see the in-game result

We end with the Noise modifier to have some rocky vertex. Select the produced mesh. In the modifier Plugin select 'Noise' and modify the value to have someting coherent. You now have a nice Tunnel with mapping coordinates you can tweak in the 'UV Parameters'.
