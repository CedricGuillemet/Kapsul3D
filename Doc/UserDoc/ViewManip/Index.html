View Manipulation

As for the standard operation, there is also an undo for view manipulation. It works like a standard undo/redo and apply to the view. This menu is accessible from the 'View' menu item. You can see the shortcuts on this screenshot. View Undo/Redo works on any views and type of views.

Please note that Kapsul 3D needs a 3 buttons mouse!

 

a) Camera/Perspective Views

 

To move the Camera/Perspective target, set the camera you want to tweak as the active view. Press down the middle mouse button and drag the mouse.

 

To move the Camera/Perspective Up/Down and Left/Right. Press CTRL+ALT, press down the middle mouse button and drag the mouse. Release the mouse button to stop moving the camera.

With the keyboard, you can move up/down with PGUP/PGDOWN keys. Strafing left/right with the left/right arrows.

 

Same as above but CTRL+SHIFT and middle mouse button. Moving Forward/Backward can also be accomplished by the wheel mouse.

With the keyboard, you can move forward/backward with UP/DOWN arrows. Strafing left/right with the left/right arrows.

 

To rotate the camera around its target, set the camera as the active view, press CTRL and the middle mouse button and drag the mouse.

The other way is to click the button and left-click in the camera view. Then drag the mouse.

 

Rotating the camera around its target can help u a lot for seeing a mesh from every angle. In this example, we want to rotate around the cube and sphere couple.

 

We select the two meshes.

 

Press the 'W' key or go to the view menu and select 'Hook Camera Target':

The camera target is now set at the center of the selected mesh.

You can now rotate around those mesh using CTRL and the middle mouse button.

2) Zooming

 

Click that icon to activate the fast zooming function.

All the ways for zooming are:

- For fast zooming feature : click in a ortho view, let the left mouse button down and drag.

- With keyboard: HOME zoom out, END zoom in.

- With wheel mouse

3) Region zoom

 

Click that icon to activate the region zoom function.

Basically, this function works that way: you select a rectangle in a view then, that view is zoomed in order to show you all the stuff in the rectangle area.

 

I build a rectangle.

 

The view is then zoomed to show was is in my rectangle area.

 

 

 

 

4) Centering views

 

Those 2 icons rescpectively : center the current Ortho view , center all the Ortho views.

Centering a view means zooming and paning a view automaticaly to make everything from the scene visible.

 

From anywhere you are in the scene.

 

Click the center icon and all the scene is visible.

e) View Shortcuts

 

CTRL+F

Set/unset a view at full window size.

 

 

T

Set the active view as TOP ortho

 

 

F

Set the active view as FRONT ortho

 

 

K

Set the active view as BACK ortho

 

 

L

Set the active view as LEFT ortho

 

 

R

Set the active view as RIGHT ortho

 

 

B

Set the active view as BOTTOM ortho

 

 

C

Open the CAMERA selection dialog from which you can choose a camera
   

U

Set the active view as USER ortho (Previous Ortho layout)
   

V

Set the active view as UV MAPPING View

 

 

PGUP

Move the active camera/perspective UP

 

 

PGDOWN

Move the active camera/perspective DOWN

 

 

ARROWS

Move the active camera/perspective left/right/up/down correspondong to key stroke.

 

 

P

Set the active view as PERSPECTIVE

 

 

Z

Enter/Exit FREE FLY mode
   

S

Enable/Disable SNAp to grid

 

 

S

Move forward (FREE FLY mode)
