Skybox 

Create/modify a skybox from the 'tools' menus. Select 'Skybox Editor'

Skybox can help you design a map by giving a real immersion and feeling. In Kapsul, the skybox works that way:

  • create a skybox in the skybox editor
  • Load it in a camera view

The skybox is a first try to make a real atmosphere in the scene. Next challenge will be support for clouds, fogs, sun(s),…

 

a) Skybox overview

A skybox is composed of 6 images making a cube around the world. The matrix is a little different from the Camera matrix. It reflects only the rotation, so you can turn in the world but you can't touch the box.

 
   

   

 

b) Building a Skybox

The window is composed of an autopreview, the buttons to load bitmaps and a skybox file management. I won't speak of the second layer or the clouds. It's premiliary development. At this time, the skybox is empty. You can start a new one or load an already existing skybox

When you click on one of the 'none' or on one of the checkbox, it prompts you to load an image file.

Once the file is loaded, it's displayed in the autopreview.

Make the same way for the other.

Once your skybox is ended, you can click on the save button. You don't have to set all the 6 image to save it. You have to save it to use it in the camera views.

 

c) Using the Skybox in the camera views

Select a camera view, go the 'View' menu, 'Skybox'submenu and click Load. You can't set a Skybox in Ortho views. To remove a Skybox, click on the 'none' menu item.

It works like a 3D background: All the geometry is displayed in front of it.
