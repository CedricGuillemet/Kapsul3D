Background management 

Set the parameters from the 'tools' menus. Select 'Background/world cap'

Setting background is usefull for edting. It's different from the skybox. As the skybox is used to set the envirronement, the background helps you design maps from sketch or modeling characters from conceptual works.

 

a) Window overview

There are 3 groups for each view projection. Per ortho view, you define an image file, the image origin (X/), the world-sized image size and an opacity parameter. The top of each checkbox enable/disable the background drawing. The 'Keep proportions' checkbox let helps you keeping a good in-world size. The proportion calculation is only based upon the X parameter. It doesn't influence the 'Origin' parameters.

  

 

b) Setting parameters

We start by setting the Image file we want. As the file is loaded, the 'Top/Bottom' checkbox is enabled, The size correspond to the image's size.

We type a new X origin and a new X size. As you can see, the Y Size is updated because the 'Keep Proportions' is enabled. We change the 'Opacity' Level to 86%

 

c) Result

The given bitmap is under all. That means that you can put meshes or anything under. The Top/bottom bitmap isn't rendered in any other view than Top or Bottom. You can't define (for the moment) Background in Camera views.
