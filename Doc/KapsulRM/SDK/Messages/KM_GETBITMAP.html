KM_GETBITMAP


    Parameters :
    wParam : 0
    lParam : 0

    Return values :
    Return HBITMAP corresponding to your Bitmap.

    Remarks :
    This bitmap is drawn in the " OutLook-like " bar.
    Just load the DIB and return it.

    CBitmap m_Dib;
    ...
    case KM_GETBITMAP:
    value=(unsigned long)&m_Dib.LoadBitmap(IDB_BITMAP);
    break;
