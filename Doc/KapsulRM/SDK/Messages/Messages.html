Messages  
    Messages are sent to the plugin via a PROCESS procedure.
    It's very similar the way Microsoft Windows(TM) routs its message.
    in the PROCESS procedure, a switch determines the action to do.
    A int is returned in most cases.
    You don't need to handle messages you don't have to.
    Special I/O messages are unneeded in Modifier plugins
    Or, if you don't need animation, don't handle Animations GUI messages.
