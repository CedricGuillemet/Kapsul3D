KM_SETDESK


    Parameters :
    wParam : Desk pointer
    lParam : 0

    Return values :
    Nothing

    Remarks :
    This message is sent very frequently (95% of time, the desk remain the same).
    However, you can set your Desk every time or perform specific code when the Desk change :

    case KM_SETDESK:
    if ((Desk*)wParam!=dk)
    {
      dk=(Desk*)wParam;
      pd=dk->AddPrivate("SKELETTON.DLL");
      LoadPrivate();
      ListSkelettons();
    }
    break ;

    this example search over the desk (when it changes) if it has Private Datas
