KM_OPEN

    Parameters :

    wParam : HWND handle of the Plugin dialog container
    lParam : 0

    Return values :
    You must return the HWND of the dialog you just created.

    Remarks :
    This message is sent when Kapsul wants you to display a dialog. You can perform specific stuffs.
    Plugins built with the wizard create the dialog. Otherwise, check a plugin to get all the « must be set » parameters.

    MFC code :

    case KM_OPEN:

    tmphInst=AfxGetResourceHandle();
    AfxSetResourceHandle(theApp.m_hInstance);
