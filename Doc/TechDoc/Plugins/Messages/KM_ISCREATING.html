KM_ISCREATING


    Parameters :
    wParam : 0
    lParam : 0

    Return values :
    Return TRUE if you're performing stuffs.

    Remarks :
    By convention (and easier performance), you cannot change the
    working view when you're doing your stuffs. An easy example is
    that your plugins performs translating vertex. When you start draging,
    set this flag to TRUE. So the user cannot change view. Set flag to FALSE
    when releasing the mouse.
