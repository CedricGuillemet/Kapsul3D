KM_KEYDOWN


    Parameters :
    wParam : MAKEDWORD(X,Y)
    lParam : Vertex *

    Return values :
    Nothing

    Remarks :
    This message corresponds to Left Mouse Button pressed Down.
    If your plugin supports mouse, the Wizards generates calls to a function
    decrypting messages values.

    case KM_LMOUSEDOWN:
    LMouseDown(LOWORD(wParam),HIWORD(wParam),(Vertex *)(lParam));
    Break ;

    The LMouseDown function is declared like this

    void LMouseDown(int X,int Y,Vertex *Pos)

    X and Y are pixel coordinates.
    Pos is a pointer to a " Vertex " structure. The pointed vertex belong to
    a Zero base surface corresponding to the X,Y value in the current " View "
